## Description

생활용품 판매 웹사이트 공백을 모티브로 한 팀프로젝트

<br />

## What did I do

&emsp;CRA를 이용한 프로젝트 초기 세팅  
&emsp;Sass 를 활용한 레이아웃 디자인  
&emsp;Javascript, CSS를 이용한 슬라이드 메인 배너 구현  
&emsp;상품 리스트와 각 상품을 컴포넌트로 분리해 코드 재활용  
&emsp;로그인 토큰 유무에 따른 회원가입 유도 팝업 구현  
&emsp;Javascript, CSS를 이용한 모달 구현  
&emsp;formData를 이용한 multiple file업로드 기능  
&emsp;Slack, Notion, Trello를 이용한 scrum 방식 아래 프로젝트 진행

<br />

## Tech Stack

HTML/CSS, Sass, Javascript(ES6+), React, Git
